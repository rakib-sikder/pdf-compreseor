 # PDF Compressor

This project is for compressing PDF files and reducing their size.

## Features

- Compress PDF files
- Reduce file size
- Keep documents readable
- Simple workflow

## Usage

1. Open the project.
2. Select a PDF file.
3. Run the compression step.
4. Save the compressed PDF.

## Notes

- Results may vary based on the source PDF.
- Keep a backup of the original file.
